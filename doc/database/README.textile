h1. Database Designer - Getting Started

To start using the modeler, here is a quick start procedure:

*1) Create a modeling project*

* make sure you are in the Modeling perspective
* then click on "New / Modeling Project"

!images/NewModelingProject.png!

*2) Create a database model*

* right click on theÂ modeling project and select the menu "New / Other..."
* select "Other / Database Model"

!images/NewDatabaseModel.png!

*3) Activate the "database" viewpoint*

* right click on the modeling project and select the menu "Viewpoint selection"
* select database in the dialog

!images/ViewpointSelection.png!

*4) Create a diagram*

* right-click on the database object from your database model
* select the menu "New representation / XXX database"

!images/CreateDatabaseDiagram.png!

*5) Use the palette to populate the diagram*

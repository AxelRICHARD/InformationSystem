Services

Name Returned type M2Doc Syntax Description Example
getAllNamespaces List of Namespace {m:myNamespacesContainer.getAllNamespaces()} Returns the list of Namespace directly and indireclty contained in the given NamespacesContainer. {m:myNamespacesContainer.getAllNamespaces().name->sep(',')} returns: A comma separated list of the given NamespacesContainer's Namespaces name.
getName String {m:myNamespace.getName()} Returns the name of the given Namespace. {m:myNamespace.getName()} returns: 'mynamespace'
getOwnedDTO List of DTO {m:myNamespace.getOwnedDTO()} Returns the list of DTO directly contained in the given Namespace. {m:myNamespace.getOwnedDTO().name->sep(',')} returns: The list of DTO directly contained in the given Namespace names separated by coma.
getOwnedEnumerations List of Enumeration {m:myNamespace.getOwnedEnumerations()} Returns the list of Enumeration directly contained in the given Namespace. {m:myNamespace.getOwnedEnumerations().name->sep(',')} returns: The list of Enumration directly contained in the given Namespace names separated by coma.
getOwnedReferences List of Reference {m:myNamespace.getOwnedReferences()} Returns the list of Reference directly contained in the given Namespace. {m:myNamespace.getOwnedEnumerations().name->sep(',')} returns: The list of Reference directly contained in the given Namespace names separated by coma.
hasAllNamespaces boolean {m:myNamespacesContainer.hasAllNamespaces()} Returns true if the given NamespacesContainer contains at least one Namespace. {m:myNamespacesContainer.hasAllNamespaces()} returns: true or false.
hasOwnedDTO boolean {m:myNamespace.hasOwnedDTO()} Returns true if there is at least one DTO directly contained in the given Namespace, false otherwise. {m:myNamespace.hasOwnedDTO()} returns: true or false.
hasOwnedEnumerations boolean {m:myNamespace.hasOwnedEnumerations()} Returns true if there is at least one Enumeration directly contained in the given Namespace, false otherwise. {m:myNamespace.hasOwnedEnumerations()} returns: true or false.
hasOwnedReferences boolean {m:myNamespace.hasOwnedReferences()} Returns true if there is at least one Reference directly contained in the given Namespace, false otherwise. {m:myNamespace.hasOwnedEnumerations()} returns: true or false.
qualifiedName String {m:myNamespaceContainer.qualifiedName()} Returns the qualifiedname of the given NamespaceContainer. {m:myNamespace.qualifiedName()} returns: a.b.mynamespacecontainer

Services and attributes for the Property concept

Services

Name Returned type M2Doc Syntax Description Example
getName String {m:myProperty.getName()} Returns the name of the given Property. {m:myProperty.getName()} returns: 'myReference' or 'myAttribute'
getOriginTypeIfDifferent String {m:myProperty.getOriginTypeIfDifferent(someStructuredType)} Returns a String representing the origin StructuredType of the given Property if its differente from the given context StructuredType. {m:myProperty.getOriginTypeIfDifferent(someStructuredType)} returns: 'From supertype namespace1.someDTO' or 'From Entity namespace1.someEntity' or 'From DTO namespace1.someDTO' or ''.

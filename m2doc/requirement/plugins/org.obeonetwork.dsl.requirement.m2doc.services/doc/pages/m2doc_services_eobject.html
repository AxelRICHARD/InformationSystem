Services and attributes for the EObject concept

Services

Name Returned type M2Doc Syntax Description Example
allRelatedRequirements List of Requirement {m:obj.allRelatedRequirements()} Returns a list of all the Requirements related to the target object or one of its sub objects. The result is a list with no duplicates sorted against the requirements ids. {m:obj.allRelatedRequirements()->sep(',')} returns: A comma separated list of related requirements.
hasRelatedRequirements boolean {m:obj.hasRelatedRequirements()} Returns true if the given Object as at least one related Requirement. {m:obj.hasRelatedRequirements()} returns: true or false.
relatedRequirements List of Requirement {m:obj.relatedRequirements()} Returns the set of Requirement that are related to the target object. {m:obj.relatedRequirements()->sep(',')} returns: A comma separated list of related requirements.
relatedRequirementsWithSubtype List of Requirement {m:obj.relatedRequirementsWithSubtype('SUBTYPE')} Returns the list of Requirements that reference the target object and that have the specified subtype. {m:obj.relatedRequirementsWithSubtype('performance')->sep(',')} returns: A comma separated list of related requirements of performance subtype.

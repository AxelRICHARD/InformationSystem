To run the sample, you can deploy the org.obeonetwork.informationsystem.gen.web.sample project in a webServer like Eclipse orion.
You can also use nodeJs :
First, install nodeJs on your computer and check that "node" is in your terminal command path.
After that, set your terminal context in this project en write the command line :
	npm install connect
Now, nodeJs is ready to work. To start a server, type :
	node nodeJsRunner
You can open your favorite browser with this URL : http://127.0.0.3:8080/
